スキル

HTML5

HTMLは主要なタグを理解し、デザインに合わせたマークアップをSEOの観点から正しくタグ付けします。

CSS

レスポンシブ対応が可能です。アニメーションなど複雑な動きは調べることで実装可能です。

SCSS

変数やネスト構造、mixin機能を使用した記述ができます。2作目以降からSCSSを使用して作成しています。

jQuery

jQueryを使用した簡単なコードを書くことができます。変数、四則演算、条件文、繰り返し文などの基本構造がわかります。また、Parallax,slick,easingは制作物に実装しています。（bxslider、ハンバーガーメニュー、lightbox2）

Javascript

jQueryでJavascriptに触れたため、現在勉強中です。

Photoshop

色調補正やパスの切り抜き、解像度の変更、画像の出力など基本的な処理を行えます。制作したバナーやWEB作品で使用している写真すべてをPhotoshopで加工しています。

Illustrator

図形の作成や文字組など、基本的な内容を行えます。履歴書、職務経歴書はillustratorで作成しています。

WordPress

固定ページの編集や記事の投稿、固定ページの作成と、更新業務を行えます。

Figma/XD

サイトのワイヤーフレームとデザインカンプを作成できます。プロトタイプは画面推移や追尾など、簡単な動作を実装できます。1作目はXD、2作目以降はFigmaで制作しています。

Git・GitHab

マージ、コミットなど基本的な作業を行えます。

Word

請求書、納品書などのビジネス文書作成、差込印刷などを行えます。

Excel

リスト作成や基本的～応用の関数、ピボットテーブルの作成などを行えます。

PowerPoint

プレゼン資料や研修資料の作成を行えます。

佐々木　愛

コーディングも画像加工もやりごたえがあり楽しく感じます。現在PHP、jQuery、Javascriptを勉強中です。
アニメーションももっと制作してみたいです。
デザインやアートに強い関心があります。好きなものは美術館や博物館、空・植物・動物、ゲーム・漫画・小説などです。集中していると時間を忘れて没頭して作業しています。
身近にデザインやアートを仕事で扱う人が数名居り、年々興味が強くなってきました。前職の出版社でさらに興味を持ちました。今後はより制作に関わっていけたら幸いです。
東京都板橋区在住。アジャストアカデミー在校中です。
